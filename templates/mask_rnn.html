IMPLEMENTATION OF MASK R-CNN ARCHITECTURE ON A CUSTOM DATASET

PROBLEM

The goal of this project is to detect objects and generate boundary boxes in an image.


SOLUTION

Implemented Mask-RCNN architecture and pretrained weights to generate boundary boxes for my own collection of images

ANALYSIS

  • First clone the Mask RCNN repository which has the architecture for Mask-RCNN model from this link
  • Download the pretrained weights for training the model using this link
  • Using OpenCV in python and other libraries, the scope extends to detecting the objeccts in a live video
